# jovonbuilds Claude Code Plugins

A plugin marketplace for [Claude Code](https://claude.com/claude-code).

## Install the marketplace

```bash
claude plugin marketplace add jovonbuilds/claude-plugins
```

## Available plugins

| Plugin | Description |
|--------|-------------|
| [claude-needs-you](https://github.com/jovonbuilds/claude-needs-you) | Native OS notifications when Claude Code needs your attention |

## Install a plugin

```bash
claude plugin install claude-needs-you@jovonbuilds
```
